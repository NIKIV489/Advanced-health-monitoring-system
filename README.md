# Advanced System Health Monitoring Dashboard Using Python

## Objective
To monitor system health parameters such as CPU usage, RAM usage, battery status, and running processes in real time using Python.

## Features
- Real-time CPU and RAM monitoring
- Battery status detection
- Lists all running processes sorted by CPU usage
- Web-based dashboard using Flask
- Auto-refresh every few seconds

## Technologies Used
- Python
- Flask
- psutil
- HTML & CSS
